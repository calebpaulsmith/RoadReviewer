FHWA Road Checker (provisional title)

Federal-aid road classification — FHWA functional class + 2020 Adjusted Census Urban Boundary. Unofficial tool; not affiliated with FHWA or FEMA.
Nothing you paste is uploaded or saved. This page has no server and no analytics. Your browser queries the public state-DOT and USDOT map services directly (the same services the RoadReviewer Excel tool queries) and shows every request it makes in the network log at the bottom. Refreshing the page erases everything.
Accepts lat, lon with an optional name before or after — commas, tabs or spaces. Classification runs as you paste; no submit button.

This tool classifies the road, never the project — a "Federal aid" tag means the road is on the federal-aid system (Urban Minor Collector or greater per FHWA functional classification and the 2020 Adjusted Census Urban Boundaries); the inspector or applicant decides what that means for any given work order. Road-class lookup is wired for Michigan (MDOT), Indiana (INDOT) and Wisconsin (WisDOT); other states get the urban-boundary check only. Map tiles © Esri World Imagery / OpenStreetMap contributors — viewing the map fetches tiles for the visible area from those providers.
